# eval_site_bootstrap -- Reproduction de site en Bootstrap

Exercice d'evaluation : reproduire le plus fidelement possible un site exemple en 2h chrono.

Demo live : [chimerical-croissant-933730.netlify.app](https://chimerical-croissant-933730.netlify.app/)

---

## Lancement

Ouvrir **index.html** directement dans le navigateur -- aucune installation requise.

---

## Stack technique

* **HTML / CSS** -- structure et styles
* **Bootstrap** -- mise en page, composants et responsive via CDN
* **Netlify** -- deploiement 

---

## Structure des fichiers
```
eval_site_bootstrap/
├── index.html      <- Page unique du site
├── style.css       <- Surcharges CSS personnalisees par-dessus Bootstrap
└── images/         <- Assets visuels du site
```

---

## Contexte

Evaluation realisee en conditions contraintes (2h). L'objectif etait de reproduire
fidelement la structure, le design et le contenu d'un site de reference en utilisant
Bootstrap comme base, complété par des styles CSS personnalises.
